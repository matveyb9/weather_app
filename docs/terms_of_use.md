## 2. Описание сервиса

Приложение «Погода» предоставляет информацию о текущей погоде и прогнозе для любого города мира. Данные предоставляются сторонними открытыми сервисами:

- **Open-Meteo** (api.open-meteo.com) — данные погоды и геокодирование
- **Nominatim / OpenStreetMap** (nominatim.openstreetmap.org) — обратное геокодирование

Приложение является бесплатным и не требует регистрации или создания учётной записи.

---

## 3. Использование данных о местоположении

Приложение может запрашивать доступ к GPS-координатам устройства исключительно для определения текущего местоположения и получения прогноза погоды для него.

- Координаты передаются напрямую в API Open-Meteo и Nominatim для получения данных о погоде и названия населённого пункта.
- Координаты не хранятся на серверах разработчика и не передаются третьим лицам, за исключением указанных API-сервисов.
- Разрешение на геолокацию является опциональным. Приложение полностью функционирует при ручном вводе города без предоставления доступа к местоположению.

- точность, полноту или актуальность прогноза погоды;
- непрерывную доступность сервиса;

Не принимайте ответственных решений, основываясь исключительно на данных Приложения.
